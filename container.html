STEP 03

백엔드 컨테이너화 (Docker)

"내 컴퓨터에서는 되는데?" 문제를 해결하고 Scale-out이 안되는 환경에서 탈피하기.

해야 할 것들 (To-Do)

  • Dockerfile 작성하기

    Node.js나 Python 서버의 실행 환경 명세서를 Dockerfile로 작성해 이미지를 만듭니다.

  • 로컬에서 docker-compose 실행

    웹 서버뿐 아니라 포스트그레스(PostgreSQL)나 레디스(Redis) 같은 DB를 컨테이너로 한 번에 띄워서 테스트하는 Compose 파일을 만듭니다.

  • 컨테이너 환경(AWS EC2) 등에서 실행

    개발된 이미지를 Docker Hub나 ECR에 올려두고, 실제 운영 서버에서 Pull하여 Daemon 모드로 실행해봅니다.

추천 기술 스택
