STEP 04

인프라 코드 관리 (IaC)

개발을 위한 AWS 콘솔의 수동 프로비저닝을 벗어나 선언형 아키텍처로 진화하기.

해야 할 것들 (To-Do)

  • Terraform CLI 설치 및 AWS 연동

    AWS IAM 사용자를 생성하고, Terraform을 활용해서 AWS Provider를 설정 및 인증합니다.

  • 간단한 S3 버킷 및 보안그룹 생성

    콘솔 접속 없이 설정 파일(.tf)만을 이용해 클라우드에 S3와 방화벽 설정(Security Group)을 띄워봅니다.

  • Terraform State 원격 관리 (심화)

    로컬에 저장되는 tfstate 파일을 AWS S3 백엔드로 옮겨 여러 개발자가 인프라를 협업할 수 있도록 구성합니다.

추천 기술 스택

Terraform AWS IAM AWS S3 / VPC
